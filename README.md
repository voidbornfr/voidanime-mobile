## ✨ The Vision
**VoidAnime** is not just another streaming app. It's a design-first experience built for anime enthusiasts who appreciate a clean, minimal, and modern aesthetic. Taking inspiration from **monochrome glassmorphism**, the app provides a premium interface that stays out of your way while you dive into your favorite stories.

---

## 🚀 Key Features

- 🌑 **Monochrome Glassmorphic UI**: A consistent, sleek black-and-white theme with frosted glass effects.
- 🔄 **Dual API Synchronization**: Seamlessly switch between **AniList** and **MyAnimeList** (MAL) for progress tracking.
- 📡 **Multi-Provider Streaming**: Integrated with premium sources like **HiAnime**, **Animelok**, and more.
- 📥 **Advanced Downloads**: Native support for **Torrent** and **CDN-based** downloads for offline viewing.
- 📖 **Manga Support**: Built-in high-performance manga reader for on-the-go reading.
- 📰 **Anime News**: Stay updated with the latest happenings in the anime world directly from the app.
- ⚙️ **Deep Personalization**: Highly customizable settings, from playback controls to theme fine-tuning.

## 🛠️ Technology Stack

- **Core Framework**: [Flutter](https://flutter.dev) (for high-performance cross-platform UI)
- **Backend/Logic**: [Go (Golang)](https://golang.org) (integrated via mobile-bridge for proxying and heavy lifting)
- **State Management**: Provider
- **Storage**: SQLite & Shared Preferences
- **Networking**: Dio & Shelf
